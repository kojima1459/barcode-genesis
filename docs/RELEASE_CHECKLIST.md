# リリース前チェックリスト

出荷前に以下の項目を手動で確認してください。

---

## 認証フロー

- [ ] **ログイン**: Google ログイン → ホーム画面に遷移する
- [ ] **ログイン維持**: リロード後もログイン状態が維持される
- [ ] **ログアウト**: プロフィール → ログアウト → ログイン画面に戻る

---

## 図鑑（Dex）

- [ ] **タブ切り替え**: ALL / DRINK / SNACK / DAILY / BEAUTY / OTHER タブが正常に動作
- [ ] **日本語表示**: タブラベルとロボット情報が日本語で統一されている
- [ ] **Safari翻訳誘発なし**: iOS Safari で「翻訳しますか？」が表示されない

---

## ショップ（Shop）

- [ ] **商品表示**: コイン商品カードのタイトル/説明が日本語で表示される
- [ ] **英語残留なし**: "Coin Bag" / "Coin Pouch" 等の英語が残っていない
- [ ] **購入フロー**: 商品タップ → 確認ダイアログ → 購入処理が動作する

---

## プロフィール（Profile）

- [ ] **アバター保存**: ヘッダー左の画像を変更 → 保存後に反映される
- [ ] **ユーザー名変更**: 名前変更 → 保存後に反映される

---

## プレミアム（Premium）

- [ ] **表示**: プレミアム画面が正常に表示される
- [ ] **登録ボタン**: 押下 → SYSTEM ERROR にならない
- [ ] **Stripe連携**: 課金導線が正しく遷移する（または期待されるエラー表示）

---

## ボス/ミッション

- [ ] **デイリーボス**: ログイン後に表示 → エラーなし
- [ ] **週間ボス**: ログイン後に表示 → エラーなし
- [ ] **デイリーミッション**: 一覧表示 → エラーなし
- [ ] **ミッション報酬**: 受け取り → エラーなし

---

## バトル

- [ ] **対戦開始**: ユニット選択 → マッチング → バトル開始
- [ ] **バトル再生**: アニメーションが正常に再生される
- [ ] **結果表示**: 勝敗/報酬が正しく表示される

---

## 全般

- [ ] **iOS Safari**: PWA として動作確認
- [ ] **ランタイムエラー**: コンソールに SYSTEM ERROR が出ない
- [ ] **翻訳キー漏れ**: 【未翻訳:xxx】が表示されない（本番環境）
